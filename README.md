## Getting Started 

Add NEXT_PUBLIC_API_NINJAS_KEY to .env file to use search.

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
